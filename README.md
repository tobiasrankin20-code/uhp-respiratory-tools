# UHP Respiratory Clinical Tools

**Version 1.0 — Working Draft — May 2025**

A Progressive Web App (PWA) providing evidence-based clinical decision support for the Acute Respiratory Assessment Unit (ARAU) and Respiratory Virtual Ward at University Hospitals Plymouth NHS Trust.

---

## ⚠️ Important — Working Draft

**This tool has not yet received formal clinical approval and should not be used to guide clinical decisions without consultant oversight.** It is currently in development and undergoing review by the responsible respiratory consultant before wider implementation.

---

## Tools Included

| Tool | Guideline |
|---|---|
| COPD Early Supported Discharge | DECAF · PEARL · NHS England VW Guidance 2022 |
| Acute NIV Decision Tool | UHP SOP Dec 2023 · BTS/ICS 2016 |
| Acute Asthma Severity | BTS/SIGN 2024 · NICE NG244 |
| Oxygen Therapy Prescriber | BTS Emergency Oxygen 2017 |
| PE Risk Stratification | Wells · PERC · sPESI · NICE NG158 |
| Clinical Frailty Scale | Rockwood · BGS 2020 |
| COPD Frequent Attender Profile | PEARL · GOLD 2025 |
| CAT Score | Jones et al ERJ 2009 · GOLD 2025 |
| ABG Interpreter | Standard acid-base physiology · BTS/ICS 2016 |
| DECAF Score | Steer et al Thorax 2012 |
| MRC Dyspnoea Scale | Fletcher 1959 · eMRCD · DECAF |
| Spirometry Interpretation | ARTP 2020 · BTS 2019 · GLI 2012 · GOLD 2025 |
| ECOG Performance Status | Oken et al 1982 · WHO |

Each tool has an accompanying evidence summary accessible within the app.

---

## Technical

- Single HTML file — no dependencies, no external data connections
- Works offline once loaded (Progressive Web App with service worker)
- No patient data is collected, stored, or transmitted
- Installable on Android (Chrome) and iOS (Safari)
